About Mold Theme

Mold Theme is Html Template, build for personal use and base theme structure to start with.

Features

Personalized Template
Mold come with great color scheme changer. which is handy to use.
Responsive Design
Mold is resposive, no matter what device phone, table or desktop.
Reusable Component
There are resusable component that can be used in multiple page or remove them easily
Custom Icon Fonts
you will get 263 costom designed icon fonts that is more easier to use.
Based on Bootstrap 3
Mold is built above bootstrap, means features of bootstrap can be used in addition.
Built with less/kit
Built with Less/kit. You will get built files along with compiled files
Fast To Launch
Its designed to built website fast and robust.
Love to Simplicity
Build simple, easier to manage and intergrate to any framework or cms.
